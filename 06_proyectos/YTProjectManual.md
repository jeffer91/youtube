# Bloque 06 — Proyectos

Objetivo: crear un sistema local de proyectos para AutoEdit Studio.

Guarda proyectos en user_data/projects y mantiene:

- user_data/database/YTProjects.json
- user_data/database/YTCurrentProject.json

Prueba:

1. npm install
2. npm start
3. Seleccionar video MP4
4. Crear proyecto
5. Revisar user_data/projects/project_.../YTProject.json
6. Ejecutar diagnóstico

Terminal:

npm run check:projects

Aprobación: se crea proyecto, índice, sesión actual y diagnóstico reconoce Bloque 06.
