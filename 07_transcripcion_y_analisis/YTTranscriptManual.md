# Bloque 07 — Transcripción y análisis

Objetivo: guardar una transcripción editable dentro del proyecto actual y generar un análisis base para preparar clips.

Archivos:
- YTTranscriptModel.js
- YTTranscriptStore.js
- YTAnalysisService.js
- YTTranscriptService.js
- YTTranscriptIpc.js
- YTTranscriptCheck.js
- YTTranscriptManual.md

Prueba:
1. npm install
2. npm start
3. Seleccionar video
4. Crear proyecto
5. Pegar texto en transcripción editable
6. Guardar transcripción
7. Analizar texto
8. Revisar resumen, palabras clave y segmentos sugeridos
9. Ejecutar diagnóstico

Terminal:
npm run check:transcript

Aprobación: se crean YTTranscript.json, YTAnalysis.json y YTTranscriptSession.json.
