# Bloque 4 - Entrega final

Este documento resume el Bloque 4 en lenguaje simple.

## Objetivo del bloque

Dejar la app lista para uso diario, verificacion y creacion de instalador.

## Que se agrego

| Parte | Estado |
|---|---|
| Carpeta entrega | Creada |
| Conexion de entrega | Creada |
| Verificacion bloque 4 | Creada |
| BAT para abrir app | Creado |
| BAT para actualizar y abrir | Creado |
| BAT para verificar app | Creado |
| BAT para crear instalador Windows | Creado |
| Configuracion electron-builder | Agregada |
| Guia de uso diario | Agregada |

## Archivos BAT principales

```txt
abrir_app.bat
actualizar_y_abrir.bat
verificar_app.bat
crear_instalador_windows.bat
```

## Comando de revision

Desde Visual Studio Code:

```bash
npm run check:bloque4
```

## Crear instalador

Desde Visual Studio Code:

```bash
npm run dist:win
```

O con doble clic en:

```txt
crear_instalador_windows.bat
```

## Estado final del bloque

Bloque 4 completado a nivel de codigo y organizacion.

La app queda preparada para abrirse, actualizarse, verificarse y generar instalador de Windows.
