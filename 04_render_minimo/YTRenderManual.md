# Bloque 04 — Render mínimo

Objetivo: cortar los primeros 5 segundos del video cargado y exportar un MP4 de prueba.

Prueba:

1. Ejecutar npm install.
2. Ejecutar npm start.
3. Seleccionar un video MP4.
4. Presionar Renderizar prueba.
5. Confirmar que exista user_data/exports/render_prueba.mp4.
6. Presionar Diagnóstico y verificar Bloque 04.

Criterio de aprobación: el render se crea, el archivo se abre y el diagnóstico reconoce el Bloque 04.
