Cómo funciona Video Editor

Guía rápida para entender el flujo completo de la app por pasos y capas.

1. Idea principal

La app edita videos por etapas. Cada pantalla hace una tarea concreta y guarda el avance dentro del proyecto activo.

El video original no debe dañarse. Las mejoras se guardan como resultados nuevos o capas del proyecto.

2. Flujo recomendado

  1. Cargar proyecto y videos.
  2. Mejorar audio.
  3. Transcribir video.
  4. Crear subtítulos.
  5. Detectar silencios.
  6. Planificar edición con IA.
  7. Cortar y ordenar partes.
  8. Agregar audio, música, textos, imágenes y efectos.
  9. Aplicar estilo visual, color, formato y limpieza final.

3. Pantalla 01: Cargar

Sirve para seleccionar uno o varios videos, ordenarlos, elegir un estilo y guardar el proyecto base.

Al guardar, la app crea un archivo de proyecto y pasa a la pantalla de audio.

4. Pantalla 02: Audio

Permite mejorar voz, reducir ruido y nivelar volumen con FFmpeg desde Electron.

El resultado se compara con el original antes de guardarlo como capa o descargarlo.

5. Pantalla 03: Transcribir

Convierte la voz del video en texto limpio, segmentos con tiempo y subtítulos SRT.

Puede trabajar con texto manual TXT, SRT manual o Whisper local cuando esté instalado.

6. Motores de transcripción

  • Manual TXT: pegar texto simple para pruebas o correcciones rápidas.
  • Manual SRT: pegar subtítulos con tiempos.
  • Whisper local: transcripción real desde el equipo.

La app no debe fingir transcripciones. Si Whisper no está instalado, mostrará un aviso claro.

7. Pantallas 04 a 18

Estas pantallas están planificadas para subtítulos, silencios, plan IA, cortes, audios, efectos, textos, música, estilo, imágenes, animaciones, transiciones, color, formato cuadrado y limpieza.

Se irán activando por bloques para evitar archivos gigantes y errores difíciles de corregir.

8. Reglas de uso

  • Primero carga un proyecto antes de usar las demás pantallas.
  • No borres los videos originales mientras el proyecto esté abierto.
  • Guarda cada mejora antes de pasar al siguiente paso.
  • Revisa el resultado antes de exportar o continuar.
